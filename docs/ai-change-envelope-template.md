# AI Change Envelope Template

Use this before AI-assisted edits in the Npcink and Magick AI repository
family.

```markdown
Target repositories:
Focused module:
Intended change:
Explicit non-goals:
Boundary owner:
Public contracts touched:
Files expected to change:
Files or areas that must not change:
Required gates:
Cross-repo matrix requirement:
Rollback plan:
```

Do not start broad edits until the envelope rules out unrelated refactors,
direct WordPress writes, second control-plane ownership, and destructive Git
cleanup.
